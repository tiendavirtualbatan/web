Dominio Personalizado con los dominios de Google y las páginas de Github

Usando un Dominio personalizado con Google Dominios y Páginas Github para su sitio web/blog estático

Si este contenido lo ayuda, considere apoyar mi trabajo a través de PayPal

Siguiendo los ejemplos hasta ahora, ya tenemos nuestro sitio web alojado en las páginas de GitHub y accesible a través de Nombre de url o ??????.github.io, pero lo que realmente queremos es tener algo más personalizado, como un dominio .dev, que el Google lanzado recientemente, ¿Correcto?

Aprenderemos cómo configurar las páginas de GitHub con Google Dominios (o Google Domans con Páginas de Github).

Registrar un dominio en los dominios de Google

Acceda a la URL https://domains.google y haga clic en Buscar un Dominio (o encontrar al dominio.

Deberá iniciar sesión con su Gmail.

Los caminos de ejemplo, busquéejemplo.comEn el campo de búsqueda y google me muestra que el dominio está disponible.

Si quería registrar este dominio, simplemente haga clic en el carrito y siga la compra.

Agregar el dominio personalizado en la Página de Github

Ir a la pestaña settings del repositorio (que es su sitio web .github.io) y ve por la pantalla Hasta la sesión GitHub Pages. Coloque el dominio en el campo de dominio personalizado y haga clic en Guardar.

En mi ejemplo, tengo el aprendiz Mastery.dev, así que eso es lo que puse en este campo.

Ahora necesitará cambiar DNS en Google Domans.

Alteración o DNS no Google Domains

En la interfaz administrativa de Google Domains (haga clic en Administrar o administrar en el "My Dominios ”en los dominios de Google) para su dominio recién comprado (o no) haga clic en la pestaña DNS.

Vaya a la sesión de "recursos personalizados" o registros de recursos personalizados y agregue los hosts de GitHub con el tipo de entrada como A. El resto que puedes ir como es, solo cambiar el tipo de entrada a A y agregue las direcciones IP:

  • 185.199.108.153
  • 185.199.109.153
  • 185.199.110.153
  • 185.199.111.153

Ahora agregue elCNAME señalando el enlace desde .github.io.Será el mismo proceso de Agregar un registro de tipoA, Pero debes poner como tipo CNAME, En la primera entrada agrega un www Y luego pon el Dirección de páginas GitHub (en mi ejemplo es el aprendiz.github.io) para su repositorio en el último Aporte.

Si todo salió bien, ahora puedes acceder a tu site ¡en la nube!- Este paso puede tomar un poco (minutos), ya que los DNS pueden tardar en propagar.
